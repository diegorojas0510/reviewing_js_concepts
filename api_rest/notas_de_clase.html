¿Què es una API

  • Api: Aplication program interface / mecanismos con que los usuarios pueden interacturar a travéz de bótones menús imagenes click etc..

  • Api Rest: Este es un mecanismo de interacción de robot con otros robots ejemplo backend con backend o backend con el frontend Rest: Representecional state transfer Api Rest en definidas cuentas es conexión de apis por medio de internet http el cual no va a responder en formato json Flujo de Api rest En la siguiente imagen podemos ver como nos conectamos desde una pagina web, hacemos peticiones hacia el servidor y este nos responde con html pero para ser más interactivo tambien utilizamos js para recibir peticiones enviarlar via api manipular el DOM con su respuesta y para que persista la información lo hacemos mediante la BD.
